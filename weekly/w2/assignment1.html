Website Critique


Bad Examples of Web Design


University Mall Theatres

Uni Mall Website

Right off the bat you can tell that this websites was made by a student from high school using skills they just learned in their IT class. The first thing when you open the website is not the University Mall theatre logo nor the times, its blinding color whose hues are ramped up to the max and do not complement each other. The next thing you notice, if you managed to still have some vision after being blinded by intense color, would be that every piece of text is in a different font. Once you get passed the blinding color and off putting text choices and start to navigate the site, you notice that the webpages have no originality to them or variety, the same design is copy pasted to the next, the only difference being the image showing movie times.


Pacific Northwest X-ray Inc.

Pacific Northwest X-ray Inc.

Unlike University Mall theatres, This website doesn’t use the entire color spectrum for each different text box and design. The website still looks like a high school student designed it with a 30 min video on HTML. If I were stumbling upon this website before hand as a professional I would immediately exit thinking that It was a scam website. The color choices do not contrast well making the text very hard to read and organized poorly.


Good Examples of Web Design


HLTV.ORG

HLTV.ORG

A good example of a good website design without being too complicated is hltv. Hltv is a website used to track matches and news about popular game CSGO. The website doesn’t have any over the top design wise but has very easily able to use UI and directory. The information is neatly organized into columns with relevant information. Since the website isn’t too complicated it runs at a fast load time for any device.


GFUEL

GFUEL.com

A good example of good website design that has the fancier designs is gfuel. Gfuel is an energy supplement sold primarily online. The website features smooth pop up menus and is very user friendly with simple navigation to be able to get anywhere you need to without hassle and with a design that is visually appealing.
